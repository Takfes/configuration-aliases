# `MSYS2` and `pacman` commands reference


## MSYS2 Installation from [msys2.org](https://www.msys2.org/).

## Essential pacman Commands

| Command                                      | Purpose                                             |
|-----------------------------------------------|-----------------------------------------------------|
| `pacman -Syu`                                | Update all packages and database                    |
| `pacman -S <package>`                        | Install package                                     |
| `pacman -R <package>`                        | Remove package                                      |
| `pacman -Ss <keyword>`                       | Search all repositories for keyword                 |
| `pacman -Qqe`                                | List explicitly installed packages                  |
| `pacman -Qqe > packages.txt`                                | List installed packages for backup                  |
| `pacman -Q <package>`                        | Query if package is installed                       |
| `pacman -Ql <package>`                       | List files owned by a package                       |
| `pacman -Qii <package>`                      | Show info on installed package                      |
| `pacman -Qo <file>`                          | Find out which package owns a file                  |
| `pacman -U <filename.tar.xz>`                | Install package from local file                     |
| `pacman -S --needed - < packages.txt`        | Install packages from a file, skipping duplicates   |


## Tips for MSYS2

- Use `mingw64.exe` shell for 64-bit Windows binaries.

- Adding MSYS2 to [Windows Terminal](https://stackoverflow.com/questions/71045716/adding-msys-to-windows-terminal)

    {
        "guid": "{0caa0dad-35be-5f56-a8ff-afceeeaa6101}",
